### 截图获取方法

**iOS 模拟器：**
```bash

npx expo start --ios

# 截图快捷键：Cmd + S
# 截图保存位置：桌面
```

**Android 模拟器：**
```bash

# 截图位置：模拟器工具栏的相机图标
```

**真机：**
- iOS: 同时按 音量上 + 电源键
- Android: 同时按 音量下 + 电源键
